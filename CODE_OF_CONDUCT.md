# Contributor Covenant 行動規範

## 私たちの約束

私たちメンバー、コントリビューター、リーダーは、年齢、体格、目に見える / 見えない障害、
民族性、性自認・性表現、経験のレベル、教育、社会経済的地位、国籍、容姿、人種、宗教、
性的指向に関わらず、私たちのコミュニティへの参加を、誰にとってもハラスメントのない
体験にすることを約束します。

私たちは、オープンで、ウェルカミングで、多様で、包括的で、健全なコミュニティに貢献する
方法で行動し、相互作用することを約束します。

## 私たちの基準

私たちのコミュニティに前向きな環境をもたらす行動の例：

- 他者への共感と思いやりを示す
- 異なる意見、視点、経験を尊重する
- 建設的なフィードバックを与え、丁寧に受け入れる
- 自分の誤りに対して責任を取り、影響を受けた人々に謝罪し、その経験から学ぶ
- 個人としてだけでなく、より広いコミュニティ全体にとって何が良いかに焦点を当てる

容認できない行動の例：

- 性的な言葉や画像の使用、あらゆる種類の性的な注意やアプローチ
- 荒らし、侮辱的・蔑視的なコメント、個人攻撃や政治的攻撃
- 公的・私的なハラスメント
- 明示的な許可なしに、他者の物理的・電子的アドレスなどの個人情報を公開する
- 職場で合理的に不適切と判断されるような行動

## 執行責任

コミュニティリーダーは、許容できる行動の基準を明確化し、執行する責任があり、不適切、
脅迫的、攻撃的、または有害とみなされる行動に対して、適切かつ公正な是正措置を取ります。

## 執行範囲

この行動規範は、すべてのコミュニティスペースに適用され、また、個人が公的な場で
コミュニティを公式に代表している場合にも適用されます。

## 執行

虐待的、嫌がらせ、その他容認できない行動の事例は、執行責任者である **service@sowaretokyo.co.jp**
に報告できます。すべての訴えは迅速かつ公正に確認、調査されます。

すべてのコミュニティリーダーは、いかなる事件の報告者のプライバシーと安全も尊重する義務を負います。
