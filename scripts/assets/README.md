# DMG 背景画像

`scripts/sign-and-notarize.sh` は `dmg-background.png` がこのディレクトリにあれば
それを DMG のウィンドウ背景として使います。

## 仕様

- **サイズ**: 540 × 380（Retina で出したい場合は 1080×760 を `dmg-background.png` として保存）
- **形式**: PNG（透過なし、白背景か任意のデザイン）
- **配置**: ウィンドウは 540×380 で、アプリアイコンは `(140, 200)`、
  Applications ショートカットは `(400, 200)` に置かれます。背景は
  この座標と整合するようデザインしてください
  （例: 中央付近に矢印 → を入れて「ここにドラッグ」と示す）

ファイルが無くてもビルドは成功します。背景は単に白になります。
